NAME

SiftMotionFFmpeg.exe

DESCRIPTION

 The SiftMotionFFmpeg.exe detects image features using MoSIFT algorithm.
This program runs in a windows 32bit system.

SET_UP

1. extract SiftMotionFFmpeg.zip
2. copy all dll files into your c:\windows\system32 folder
3. if your computer does not have Microsoft Visual C++ 2008 runtime installed, install vcredist.exe

HOW_TO_USE

Make sure all the DLLs are in the current directory (or on the path).

From the windows command line run:

SiftMotionFFmpeg.exe videoInputFileName featureOutputFileName [options]

    The first argument is the input file, the second the file with the feature point descriptors 
    The MoSift output file contains 262 descriptions for each motion interest point detected. 
    The first 6 parameters at the beginning of each point description indicate the X position, y position, frame number, scale, velocity in x-dimension and velocity in y dimension of the feature point 
    The next 128 descriptors are the standard SIFT features 
    The next 128 descriptors show the optical flow in SIFT style (but not rotation invariant!)

OPTIONS

use option -h to see usage or 
	-s [int]  step size (default : 1)
	this option sets up the number of frames to skip when finding MoSift features. 
	
	-f [int]: start frame   (default : 1)
	-t [int]: end frame    (default : end of video)
	these two options set up the range that program will run.
	
	-m [int]: minimum threshold for pixel movement    (default : 1)
	-x [int]: maximum threshold for pixel movement    (default : 100)
	if one of these options are used, additional file is generated (named as "videoInputFileName.startFrame-endFrame.stepsize") 
	
	the format of the file is :
		Total frames : %d, from: %d, to: %d, stepsize: %d min_threshold: %d max_threshold: %d\n
		frame#: x-pos y-pos dx dy distance, ....
		...
		
	-c [int]: position of frame to calculate optical flow with the target frame
	            (default : step size)
	            comparing frame pos = current pos + [argument]
	            this value should not be larger than step size
